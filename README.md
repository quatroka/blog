# Quatroka | Blog

## Dependencies

* Ruby
    * Gem Bundler
    * Gem Jekyll

## Build

* Clone this repository
* Inside the repository, execute: `bundle install`
* Start the server: `jekyll serve`
* Access: `http://localhost:4000/`
